# Poincaré Maps and the Duffing Oscillator

A fascinating visual introduction to Poincaré maps!

This project explores the Duffing oscillator and illustrates the transition from regular motion to chaos.

## About the Project

The study of dynamical systems becomes much more interesting with good visualizations.

The goal here is to demonstrate the behavior of the Duffing oscillator under different conditions.

You will be able to observe in practice the moment when the system ceases to be predictable and enters a chaotic state.

## What you will find

* Scripts to simulate the oscillator dynamics.

* Generation of detailed phase space graphs.

* Precise and didactic visualizations of Poincaré maps.

* What is the Duffing Oscillator?

## How to get started

* Clone this repository to your local machine.

* Ensure you have the necessary scientific libraries installed.

* Execute the main code to generate the graphs.
